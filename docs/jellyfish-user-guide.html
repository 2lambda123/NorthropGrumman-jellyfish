Jellyfish User Guide

This book is a work in progress and is not yet complete.

This book serves as user manual for Jellyfish. Jellyfish is a collection of tools that can be used with a System Descriptor project. Readers should review Modeling with the System Descriptor before reading this book. Complete examples from this book can be found at https://github.ms.northgrum.com/CEACIDE/ceacide-books.
