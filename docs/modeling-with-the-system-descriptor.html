Modeling with the System Descriptor

This book provides an overview of the System Descriptor concept and language and is intended to be read by those that have no familiarity with the System Descriptor and have little experience with software development. Complete examples used in this book can be found at https://github.ms.northgrum.com/CEACIDE/ceacide-books.
